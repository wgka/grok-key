xAI image + video workstation

Grok Imagine Studio

一个本地网页工作台：可管理多个 xAI API Key，并调用 grok-imagine-image 和 grok-imagine-video 做图片生成、图片编辑、文本视频生成、图生视频。

图片: /v1/images/generations 图片编辑: /v1/images/edits 视频: /v1/videos/generations 视频扩展: /v1/videos/extensions 状态轮询: /v1/videos/{request_id}
Reference
依据 xAI 官方文档实现，视频生成采用两阶段流程：先提交生成请求，再轮询 request_id 获取结果。
图片模型grok-imagine-image
视频模型grok-imagine-video
部署方式本地 Python Server
